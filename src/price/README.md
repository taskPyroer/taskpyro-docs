## 版本选择建议

- **免费版**：适合个人学习和小型项目，希望体验TaskPyro基本功能的用户
- **标准版**：适合中小型团队和企业，需要完整任务调度功能和通知机制的用户
- **专业版**：适合大型企业和组织，需要分布式架构、跨平台部署和高级监控的用户

:::
